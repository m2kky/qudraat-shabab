# منصة قدرات شباب 🚀

المساحة الأمنة للشباب 
## 🌟 المميزات

### 🎯 للمتدربين
- **دورات تدريبية متنوعة** في التطوير والتصميم والتسويق الرقمي
- **ورش عمل تفاعلية** مع خبراء في المجال
- **مؤتمرات وندوات** لتبادل الخبرات
- **شهادات معتمدة** عند إكمال الدورات
- **مجتمع تفاعلي** للتواصل والتعلم

### 📊 للمدربين والإدارة
- **لوحة تحكم شاملة** مع إحصائيات مفصلة
- **إدارة الدورات** والمتدربين
- **نظام حجز متقدم** مع إشعارات
- **تقارير وتحليلات** مفصلة

## 🛠️ التقنيات المستخدمة

### Frontend
- **React 18** - مكتبة واجهة المستخدم
- **Vite** - أداة البناء السريعة
- **React Router** - التنقل بين الصفحات
- **CSS Variables** - نظام تصميم متكامل

### Backend & Integration
- **MCP Server** - خادم Model Context Protocol
- **Figma API** - تكامل مع تصميمات Figma
- **RESTful APIs** - واجهات برمجية

### Design System
- **Responsive Design** - متجاوب مع جميع الأجهزة
- **Arabic RTL Support** - دعم كامل للغة العربية
- **Modern UI/UX** - تصميم عصري وسهل الاستخدام
- **Accessibility** - قابلية الوصول للجميع

## 🚀 البدء السريع

### 1. تثبيت المتطلبات
```bash
# تثبيت Node.js (الإصدار 18 أو أحدث)
# ثم تثبيت المتطلبات
npm install
```

### 2. تشغيل المشروع
```bash
# وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# معاينة البناء
npm run preview
```

### 3. إعداد Figma Integration
```bash
# نسخ ملف البيئة
cp env.example .env

# تعديل ملف .env وإضافة:
FIGMA_ACCESS_TOKEN=your_figma_token
FIGMA_FILE_ID=your_file_id
```
